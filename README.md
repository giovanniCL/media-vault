# Media Vault

To run:
  - cd into root folder
  - docker-compose build (only for first run)
  - docker-compose up -d

Endpoints (see Postman collection):
  - POST localhost:8080/files/upload
  - GET localhost:8080/files/\<file id\>

Minio:
  - localhost:9001
  - username: minioadmin
  - password: minioadminpassword
